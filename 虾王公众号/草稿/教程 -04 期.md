# OpenClaw 技能开发：手把手教你给 Agent 加能力

## 📌 什么是技能？

技能是 OpenClaw 的核心扩展机制，让 AI 能够执行特定任务。每个技能包含：
- SKILL.md - AI 的说明书，告诉它何时使用
- 脚本文件 - 实际执行任务的代码
- 元数据 - 版本、依赖、触发词

---

## 🎯 技能结构

```
your-skill/
├── SKILL.md           # 必需：技能说明
├── _meta.json         # 必需：元数据
├── scripts/           # 推荐：脚本目录
│   └── main.sh
└── README.md          # 可选：详细说明
```

---

## 🔧 开发步骤

### 1️⃣ 创建 SKILL.md

```markdown
---
name: your-skill
description: 一句话描述技能功能
homepage: https://github.com/your/repo
---

# Your Skill

## 使用场景
- 当用户想...
- 当需要...

## 使用方法
./scripts/main.sh "参数"
```

### 2️⃣ 编写脚本

```bash
#!/bin/bash
set -e

# 检查环境变量
if [ -z "$API_KEY" ]; then
    echo "Error: API_KEY not set"
    exit 1
fi

# 带超时请求
RESPONSE=$(curl -sf --max-time 30 \
    -H "Authorization: Bearer $API_KEY" \
    "https://api.example.com") || exit 1

echo "$RESPONSE"
```

### 3️⃣ 设置权限

```bash
chmod +x scripts/*.sh
```

### 4️⃣ 本地测试

```bash
export YOUR_API_KEY="test-key"
./scripts/main.sh "test"
```

## 💡 最佳实践

1. **容错处理** - 检查环境变量、设置超时
2. **安全优先** - 不硬编码凭证
3. **清晰描述** - 让 AI 知道何时使用
4. **语义化版本** - 1.0.0 → 1.0.1 → 1.1.0
5. **文档完整** - SKILL.md 是最重要文件
